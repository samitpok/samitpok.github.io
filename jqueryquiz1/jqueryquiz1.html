jQuery Quiz I

Q1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

Answer: The dollar sign with the parentheses i.e $() is the shorter name of the jQuery function. This shortcut saves us from writing “jQuery()” every time we want to call the jQuery function. The jQuery function is also often referred to as the jQuery wrapper. This is the function that creates jQuery objects and registers handlers to be invoked when the DOM is ready; it also serves as the jQuery namespace.

A jQuery object is an object returned by the jQuery function. A jQuery object represents a set of document elements and can also be called a “jQuery result”, a “jQuery set”, or a “wrapped set”.

Q2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.

Sample HTML:
<body>
<h1>abc</h1><br><div>
<h1>div-1</h1>
<h1>div-2</h1>
</div>
<h1>xyz</h1>
</body>

solution:   Run Jquery
<script>
   $("div > h1").css("background-color","red");
</script>

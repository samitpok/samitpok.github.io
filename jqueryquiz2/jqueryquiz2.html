1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent.

//code fragment 1
$("li").each(function(idx, e) {
$(e).css(“color”, “yellow”); });
//code fragment 2
$("li").each(function() {
$(this).css(“color”, “yellow”); });
//code fragment 3
$("li").each(function(idx) {
$(this).css(“color”, “yellow”); });

Answer: All the codes are equivalent.
In the first fragment each loops through index idx and e is the element which is looped through . Here e(element) obtained from function which is looped through is used to set the value of the color.
In Second fragment index is not used but each function loops through each li and set the color using $(this).
At last 3rd fragment index is used to loop through each and each element is accessed from $(this) inside function and its color is set to yellow.
